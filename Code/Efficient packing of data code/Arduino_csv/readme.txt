How to use these codes : 

First run the python script and then upload your arduino code.
While doing this please don't open your arduino's serial monitor.

What does this codes do?
The Arduino code is used to collect the data from sensors(DHT22, Capacitive Soil Moisture Sensor, DS18B20) and compress/packs the data and 
the python script read the data from COM port, decompresses/unpacks it and saves the data in csv file.
